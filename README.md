💣 Boom Bóng Nước Mobile (Huyền Thoại Trở Lại)

Một phiên bản remake của tựa game huyền thoại Boom Online (BnB), được tối ưu hóa đặc biệt cho trải nghiệm trên thiết bị di động (Mobile First). Game được viết hoàn toàn bằng HTML5 Canvas và Vanilla JavaScript trong một file duy nhất.

🔗 Chơi ngay tại đây: https://boom-online.netlify.app

✨ Tính Năng Nổi Bật

📱 Mobile First: Thiết kế giao diện điều khiển cảm ứng (D-Pad & Action Button) mượt mà cho điện thoại.

🎮 Cross-Platform: Chơi tốt trên cả PC (Bàn phím) và Mobile (Cảm ứng).

🚀 Hiệu năng cao: Game nhẹ, tải nhanh, không cần cài đặt.

🗺️ Hệ thống màn chơi: 3 Level với độ khó tăng dần (số lượng quái, tốc độ quái).

🎁 Vật phẩm phong phú:

P (Potion): Tăng độ dài bóng nước.

B (Bomb): Tăng số lượng bóng đặt được.

S (Speed): Giày chạy nhanh.

🤖 AI Quái vật: Quái vật tự động di chuyển và đổi hướng khi gặp vật cản.

🛠️ Cài Đặt & Chạy Local

Game được thiết kế dạng Single File nên rất dễ triển khai.

Clone repository này về máy hoặc tải file index.html.

Mở file index.html bằng trình duyệt bất kỳ (Chrome, Safari, Edge...).
